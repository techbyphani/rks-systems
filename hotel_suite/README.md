# Hotel Management System

A full-stack hotel management system with Spring Boot backend and frontend (to be implemented).

## Project Structure

```
backend_hotel/
├── backend/          # Spring Boot REST API
│   ├── src/          # Source code
│   ├── pom.xml       # Maven configuration
│   └── README.md     # Backend documentation
│
└── frontend/         # Frontend application (to be implemented)
```

## Backend

The backend is a Spring Boot application with PostgreSQL database.

### Quick Start

1. Navigate to backend folder:
   ```bash
   cd backend
   ```

2. Follow the setup guide:
   - See `backend/POSTGRESQL_SETUP.md` for detailed PostgreSQL setup
   - See `backend/QUICK_START.md` for quick reference

3. Run the application:
   ```bash
   mvn spring-boot:run
   ```

The API will be available at `http://localhost:8080`

### Backend Features

- ✅ JWT Authentication
- ✅ Room Management
- ✅ Booking Management
- ✅ Guest Management
- ✅ Dashboard Statistics
- ✅ Gallery Management
- ✅ Feedback System
- ✅ Billing System
